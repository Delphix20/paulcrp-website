Hero Quote Restore Notes
========================

Purpose
-------
This file documents the Henry Ford quote that used to appear in the hero/banner area of paulcrp.com before it was replaced by the World Football Cup 2026 promo banner.

Original Quote Content
----------------------
Quote:
"Whether you think you can or you think you can't, you're right."

Author:
Henry Ford

Original Placement
------------------
The quote was inside the hero section:
section id="banner" > div class="inner"

It appeared after the main hero buttons:
- View Our Apps
- View Our Games

It used the same hero entrance animation slot:
banner-sequence banner-sequence-4

Original HTML
-------------
<div class="quote-container banner-sequence banner-sequence-4">
  <p class="inspirational-quote">"Whether you think you can or you think you can't, you're right."</p>
  <p class="quote-author">Henry Ford</p>
</div>

--------------------------

#banner .inner,
#banner .inner h2,

#banner .quote-author,

}

Font Dependency
---------------
The quote used the Allura cursive font. The page already imported it at the top:

@import url('https://fonts.googleapis.com/css2?family=Allura&display=swap');

Restore Guidance
----------------
To restore the quote later:
1. Replace or move the current promo banner from the hero area.
2. Add the Original HTML after the hero buttons.
3. Re-add the Original CSS and Original Mobile CSS.
4. Add the quote classes back to the banner color rule if needed.
